# Turismo en la Tierra Media

1. Los usuarios disponibles son Sam, Gandalf, Eowyn, Galadriel, Frodo, Aragorn y Saruman, la contraseña para todos es "1234".
2. Los usuarios Gandalf y Saruman son administradores, tanto la habilidad para ver los itinenarios de cada usuario como la de editar a cada uno se encuentran en el apartado desplegable del usuario, arriba de la opcion de logout.
3. Todos empiezan con $100 y 100 de tiempo disponible, dichos valores pueden ser modificados por los administradores.
4. En el directorio TierraMedia\src\main\resources se encuentra el archivo env.properties para cambiar el directorio en donde se localiza la base de datos.
